## Введение

Компьютеры работают с данными, представленными в виде **битов** (двоичных цифр) и **байтов**. Этот раздел объясняет, как данные кодируются, хранятся и обрабатываются с использованием различных систем счисления, а также как биты и байты формируют информацию в компьютере.

## 1. Системы счисления

Система счисления — это способ представления чисел с помощью символов (цифр) и их позиций.

- **Основные системы**:
  - **Двоичная (основание 2)**: Использует цифры 0 и 1. Пример: 1010 = 10 в десятичной.
  - **Восьмеричная (основание 8)**: Использует цифры 0-7. Пример: 12 = 10 в десятичной.
  - **Десятичная (основание 10)**: Использует цифры 0-9. Пример: 10 = 10.
  - **Шестнадцатеричная (основание 16)**: Использует цифры 0-9 и буквы A-F. Пример: A = 10 в десятичной.
- **Преобразование**:
  - **Из двоичной в десятичную**: 1010 = 1 x 2³ + 0 x 2² + 1 x 2¹ + 0 x 2⁰ = 8 + 0 + 2 + 0 = 10.
  - **Из десятичной в двоичную**: Делим на 2, записываем остатки в обратном порядке. Пример: 10 ÷ 2 = 5 (0), 5 ÷ 2 = 2 (1), 2 ÷ 2 = 1 (0), 1 ÷ 2 = 0 (1) → 1010.
  - **Двоичная ↔ шестнадцатеричная**: Группируем по 4 бита. Пример: 1010 = A (группа 1010 = 10 в десятичной = A).

> **Пример**: Число 42 в десятичной = 101010 в двоичной = 2A в шестнадцатеричной.

## 2. Биты и байты

Биты и байты — основные единицы информации в компьютере.

- **Бит**:
  - Минимальная единица информации: 0 или 1.
  - Соответствует электрическому сигналу (например, 0V = 0, 5V = 1).
- **Байт**:
  - 8 битов. Пример: 10110011.
  - Может представлять 256 уникальных значений (2⁸, от 00000000 до 11111111).
- **Большие единицы**:
  |               |            |                     |
  | ------------- | ---------- | ------------------- |
  | Единица       | Размер     | Пример              |
  | Килобайт (КБ) | 1024 байта | ~Текстовый документ |
  | Мегабайт (МБ) | 1024 КБ    | ~MP3 песня          |
  | Гигабайт (ГБ) | 1024 МБ    | ~Видео в HD         |
  | Терабайт (ТБ) | 1024 ГБ    | ~Жёсткий диск       |

> **Пример**: Символ A в ASCII занимает 1 байт (01000001).

## 3. Кодирование данных

Данные в компьютере кодируются в двоичной форме для представления текста, чисел и других типов информации.

- **Кодирование текста**:
  - **ASCII**: 7 или 8 бит на символ. Пример: A = 01000001 (65 в десятичной).
  - **Unicode**: Расширенный стандарт для поддержки всех языков (например, U+0410 для русской А).
- **Кодирование чисел**:
  - **Целые числа**:
    - Беззнаковые: 1010 = 10.
    - Знаковые (дополнение до двух): Первый бит — знак (0 = положительное, 1 = отрицательное). Пример: 1110 = -2 (в 4-битном формате).
  - **Числа с плавающей точкой** (IEEE 754):
    - Формат: знак (1 бит) + экспонента (8 бит для float) + мантисса (23 бита для float).
    - Пример: 3.14 хранится как приближение в двоичной форме.
- **Другие данные**:
  - Изображения: Пиксели кодируются как комбинации RGB (например, 24 бита на пиксель).
  - Звук: Амплитуда оцифровывается (например, 16 бит на выборку).

> **Пример**: Слово Hi в ASCII: H = 01001000, i = 01101001.

## 4. Логические операции с битами

Битовые операции манипулируют отдельными битами чисел, используются в программировании и аппаратном обеспечении.

- **Основные операции**:
  |                  |                                      |                                   |
  | ---------------- | ------------------------------------ | --------------------------------- |
  | Операция         | Описание                             | Пример                            |
  | **AND**          | 1, если оба бита 1                   | 1010 & 1100 = 1000                |
  | **OR**           | 1, если хотя бы один бит 1           | `1010                             |
  | **XOR**          | 1, если биты разные                  | 1010 ^ 1100 = 0110                |
  | **NOT**          | Инверсия бита                        | ~1010 = 0101 (в 4-битном формате) |
  | **Сдвиг влево**  | Биты сдвигаются влево, добавляются 0 | 1010 << 1 = 0100                  |
  | **Сдвиг вправо** | Биты сдвигаются вправо               | 1010 >> 1 = 0101                  |
- **Применение**:
  - **Маски**: Выделение нужных битов (например, value & 00001111 оставляет младшие 4 бита).
  - **Оптимизация**: Сдвиги вместо умножения/деления на 2 (например, x << 1 = x \* 2).
  - **Шифрование**: XOR для простого шифрования.

> **Пример**: Чтобы проверить, чётное ли число n, используем n & 1. Если результат 0, число чётное.

## Ключевые выводы

- Системы счисления (двоичная, шестнадцатеричная) — основа представления данных в компьютере.
- Биты (0/1) и байты (8 битов) — минимальные единицы информации.
- Данные кодируются в двоичной форме: текст (ASCII, Unicode), числа (целые, с плавающей точкой), изображения, звук.
- Битовые операции (AND, OR, XOR, сдвиги) используются для манипуляций с данными на низком уровне.
- Понимание битов и байтов необходимо для работы с памятью, процессорами и программированием
