## Введение

Аппаратное обеспечение (hardware) — физические компоненты компьютера, которые выполняют вычисления, хранение и передачу данных. Этот раздел объясняет ключевые элементы компьютера, включая процессор, память, хранилища, материнскую плату, периферийные устройства, а также шины и энергопотребление.

## 1. Компоненты компьютера

Компьютер состоит из нескольких основных компонентов, взаимодействующих для выполнения задач.

### 1.1. Центральный процессор (CPU)

- **Описание**: "Мозг" компьютера, выполняет инструкции программ.
- **Компоненты CPU**:
  - **ALU (арифметико-логическое устройство)**: Выполняет арифметические (сложение, умножение) и логические операции (AND, OR).
  - **Регистры**: Быстрая память для хранения промежуточных данных (например, регистр PC — счётчик команд).
  - **Управляющий блок (Control Unit)**: Координирует выполнение инструкций, управляя потоками данных.
- **Характеристики**:
  - Частота (ГГц): Скорость выполнения операций.
  - Количество ядер: Позволяет выполнять несколько задач одновременно.
- **Пример**: Intel Core i7 — многоядерный процессор с частотой ~3.5 ГГц.

> **Пример**: CPU выполняет команду ADD R1, R2, складывая значения из регистров R1 и R2 в ALU.

### 1.2. Память

Память хранит данные и инструкции, необходимые для работы компьютера.

- **Оперативная память (RAM)**:
  - **Типы**: SRAM (быстрая, используется в кэше), DRAM (медленнее, основная RAM).
  - Быстрая, энергозависимая (данные теряются при выключении).
  - Пример: 16 ГБ DDR4 RAM хранит запущенные программы.
- **Кэш-память**:
  - Расположена в процессоре (L1, L2, L3).
  - Хранит часто используемые данные для быстрого доступа.
  - Пример: L1-кэш — 64 КБ на ядро.
- **Постоянная память**:
  - **ROM**: Хранит BIOS/UEFI, энергонезависима.
  - **Flash-память**: Используется в SSD, USB-накопителях.
  - Пример: BIOS загружает операционную систему при включении.

> **Пример**: RAM хранит открытую вкладку браузера, а кэш ускоряет доступ к часто используемым данным.

### 1.3. Хранилища данных

Хранилища предназначены для долгосрочного хранения данных.

- **Жёсткие диски (HDD)**:
  - Используют магнитные диски для записи данных.
  - Медленные, но дешёвые и вместительные (например, 1 ТБ).
- **Твердотельные накопители (SSD)**:
  - Используют Flash-память, быстрее HDD.
  - Пример: NVMe SSD с скоростью чтения ~3500 МБ/с.
- **Оптические диски**: CD, DVD (устаревают).
- **Применение**: Хранение ОС, программ, файлов.

> **Пример**: ОС Windows установлена на SSD для быстрой загрузки.

### 1.4. Материнская плата

- **Описание**: Основа компьютера, соединяет все компоненты.
- **Компоненты**:
  - **Сокет**: Для установки процессора.
  - **Чипсет**: Управляет потоками данных между CPU, RAM и периферией.
  - **Слоты**: Для RAM, видеокарты (PCIe), накопителей (SATA).
- **Пример**: Материнская плата ASUS ROG поддерживает процессоры Intel и DDR5 RAM.

> **Пример**: Чипсет направляет данные от CPU к SSD через шину.

### 1.5. Периферийные устройства

- **Описание**: Устройства ввода/вывода, взаимодействующие с пользователем.
- **Типы**:
  - **Ввод**: Клавиатура, мышь, микрофон.
  - **Вывод**: Монитор, динамики, принтер.
  - **Гибридные**: Сенсорные экраны.
- **GPU (графический процессор)**:
  - Специализированный процессор для обработки графики.
  - Пример: NVIDIA RTX 4090 для игр и рендеринга.

> **Пример**: Монитор отображает интерфейс ОС, получая данные от GPU.

## 2. Шины и интерфейсы

Шины — каналы для передачи данных между компонентами.

- **Типы шин**:
  - **Адресная шина**: Передаёт адреса памяти.
  - **Шина данных**: Передаёт сами данные.
  - **Управляющая шина**: Передаёт сигналы управления.
- **Интерфейсы**:
  |           |                 |                     |
  | --------- | --------------- | ------------------- |
  | Интерфейс | Применение      | Пример скорости     |
  | PCIe      | Видеокарты, SSD | ~16 ГБ/с (PCIe 4.0) |
  | SATA      | HDD, SSD        | ~600 МБ/с           |
  | USB       | Периферия       | ~10 ГБ/с (USB 3.2)  |
- **Пример**: PCIe используется для подключения видеокарты к материнской плате.

> **Пример**: SSD передаёт данные на CPU через шину NVMe (PCIe).

## 3. Энергопотребление и охлаждение

Энергопотребление и охлаждение обеспечивают стабильную работу компонентов.

- **Блок питания (PSU)**:
  - Преобразует переменный ток (AC) в постоянный (DC).
  - Мощность: 400-1000 Вт для современных ПК.
- **Охлаждение**:
  - Воздушное (вентиляторы, радиаторы).
  - Жидкостное (водяное охлаждение).
  - Пассивное (без вентиляторов).
- **Тепловыделение**:
  - CPU и GPU выделяют тепло (до 200 Вт у мощных моделей).
  - Термопаста улучшает теплообмен.

> **Пример**: Процессор с TDP 125 Вт требует мощного кулера для предотвращения перегрева.

## Ключевые выводы

- **CPU** выполняет вычисления, используя ALU, регистры и управляющий блок.
- **Память** (RAM, кэш, ROM) хранит данные с разной скоростью и энергозависимостью.
- **Хранилища** (HDD, SSD) обеспечивают долгосрочное хранение.
- **Материнская плата** соединяет компоненты через чипсет и слоты.
- **Периферия** (клавиатура, монитор, GPU) взаимодействует с пользователем.
- **Шины** (PCIe, SATA) передают данные между компонентами.
- **Энергопотребление и охлаждение** поддерживают стабильную работу системы.
- Понимание аппаратного обеспечения необходимо для сборки ПК и оптимизации производительности.
