**–ö–æ–≥–¥–∞ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å**: –ü—Ä–æ–º—ã—à–ª–µ–Ω–Ω–æ–µ —Ä–∞–∑–≤–µ—Ä—Ç—ã–≤–∞–Ω–∏–µ, –º–æ–±–∏–ª—å–Ω—ã–µ/web –ø—Ä–∏–ª–æ–∂–µ–Ω–∏—è.


### **PyTorch** (Meta/Facebook) üî•
- **–î–∏–Ω–∞–º–∏—á–µ—Å–∫–∏–π –≥—Ä–∞—Ñ**: –ì—Ä–∞—Ñ –≤—ã—á–∏—Å–ª–µ–Ω–∏–π —Å—Ç—Ä–æ–∏—Ç—Å—è –Ω–∞ –ª–µ—Ç—É ‚Üí –ø—Ä–æ—â–µ –æ—Ç–ª–∞–∂–∏–≤–∞—Ç—å.
- **–ê–∫–∞–¥–µ–º–∏—á–Ω–æ—Å—Ç—å**: –õ—é–±–∏–º–µ—Ü –∏—Å—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª–µ–π –∏ —É–Ω–∏–≤–µ—Ä—Å–∏—Ç–µ—Ç–æ–≤.
- **–ü—Ä–æ—Å—Ç–æ—Ç–∞**: –ë–æ–ª–µ–µ –ø–∏—Ç–æ–Ω–∏—á–Ω—ã–π —Å–∏–Ω—Ç–∞–∫—Å–∏—Å.

**–ö–æ–≥–¥–∞ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å**: –ò—Å—Å–ª–µ–¥–æ–≤–∞–Ω–∏—è, –ø—Ä–æ—Ç–æ—Ç–∏–ø–∏—Ä–æ–≤–∞–Ω–∏–µ, –µ—Å–ª–∏ –Ω—É–∂–Ω–∞ –≥–∏–±–∫–æ—Å—Ç—å.

```python
import torch
import torch.nn as nn

class SimpleNN(nn.Module):
    def __init__(self):
        super(SimpleNN, self).__init__()
        self.fc1 = nn.Linear(784, 128)
        self.fc2 = nn.Linear(128, 10)
        
    def forward(self, x):
        x = torch.relu(self.fc1(x))
        x = self.fc2(x)
        return x

model = SimpleNN()
```
