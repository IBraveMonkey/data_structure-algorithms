## ğŸ“– Popular Terms

- **Pipeline**: The path code takes from the repository to the live server.
- **Artifact**: The final "file" that is actually run (e.g., a Docker image).
- **Environment**: Where the code is currently running (Dev, Test, Staging, Prod).
